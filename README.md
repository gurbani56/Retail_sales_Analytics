# Retail_sales_Analytics
End-to-end retail sales analytics project: Python-generated synthetic dataset, PostgreSQL data warehouse, SQL analytics queries, and Power BI dashboards for profitability insights and trend analysis.

**Project Overview**
This project demonstrates end-to-end data analytics workflow for a retail business, including:
- Data generation and cleaning
- Exploratory Data Analysis (EDA)
- Interactive visualizations
- SQL-based data querying and analysis

**Features**
- Data Processing Pipeline: Automated data cleaning and transformation
- Interactive Visualizations: 5 multi-panel visualization sets
- SQL Integration: PostgreSQL database with optimized queries
